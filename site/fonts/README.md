# Self-Hosted Fonts

This directory should contain the following WOFF2 font files.
Download them from the sources below and place them here before deployment.

## Required files

| File | Source | Weight/Style |
|---|---|---|
| `InstrumentSerif-Regular.woff2` | [Google Fonts](https://fonts.google.com/specimen/Instrument+Serif) | 400 normal |
| `InstrumentSerif-Italic.woff2` | [Google Fonts](https://fonts.google.com/specimen/Instrument+Serif) | 400 italic |
| `GeneralSans-Regular.woff2` | [Fontshare](https://www.fontshare.com/fonts/general-sans) | 400 normal |
| `GeneralSans-Medium.woff2` | [Fontshare](https://www.fontshare.com/fonts/general-sans) | 500 normal |
| `GeneralSans-Semibold.woff2` | [Fontshare](https://www.fontshare.com/fonts/general-sans) | 600 normal |
| `JetBrainsMono-Regular.woff2` | [JetBrains](https://www.jetbrains.com/lp/mono/) | 400 normal |
| `JetBrainsMono-Medium.woff2` | [JetBrains](https://www.jetbrains.com/lp/mono/) | 500 normal |

## Quick download

1. **Instrument Serif**: Go to https://fonts.google.com/specimen/Instrument+Serif → Download family → extract WOFF2 files
2. **General Sans**: Go to https://www.fontshare.com/fonts/general-sans → Download → extract WOFF2 files
3. **JetBrains Mono**: Go to https://www.jetbrains.com/lp/mono/ → Download → extract WOFF2 files

Until these files are placed here, the CSS falls back to system fonts (Georgia for serif, system-ui for sans, ui-monospace for mono).
